# Portfolio Website

This portfolio website showcases the talents and projects of the user, including a CV, music, engineering projects, and performances.

## Project Structure

```
portfolio-website
├── src
│   ├── assets
│   │   ├── css
│   │   │   └── styles.css
│   │   ├── js
│   │   │   └── scripts.js
│   │   └── fonts
│   └── pages
│       ├── cv.html
│       ├── music.html
│       ├── engineering.html
│       └── performances.html
├── index.html
├── README.md
└── .gitignore
```

## Features

- **CV Page**: A dedicated page to showcase the user's educational background, work experience, and skills.
- **Music Page**: A page featuring audio samples, descriptions, and links to music platforms.
- **Engineering Projects Page**: A detailed overview of engineering projects, technologies used, and outcomes.
- **Performances Page**: A showcase of performances, including descriptions, dates, and media.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the portfolio.

## Goals

The goal of this project is to create a visually appealing and informative portfolio that effectively showcases the user's skills and experiences in various fields.
